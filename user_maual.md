PyScientific Calculator - User Manual
1. Introduction
PyScientific Calculator is a powerful tool for performing numerical and symbolic mathematical calculations and visualizing functions.

2. Calculator Mode
The default tab is the calculator interface.

Input: Click buttons or type directly into the display to enter mathematical expressions.

Operations: Standard operators (+, -, *, /) and scientific functions (sin, cos, log, etc.) are available.

Evaluation: Press the = button or the Enter key to calculate the result.

Clearing: Use C to clear the entire input, and âŒ« (Backspace) to remove the last character.

3. Graphing Mode
Switch to the "Graphing" tab to plot functions.

Enter Function: In the f(x) = field, type a function of x. Examples:

x**2 + 2*x + 1

sin(x) * exp(-0.1*x)

Set Range: Specify the minimum and maximum x values for the plot axis.

Plot: Click the "Plot Function" button. The graph will appear below.

Interact: Use the toolbar above the graph to zoom, pan, and save the plot image.

4. History Panel
To view your calculation history, go to the menu bar and select View -> History Panel.

A dockable panel will appear on the right, showing a table of past expressions and their results.

Clear: Deletes all history entries.

Save CSV: Saves the current history to a calculation_history.csv file in the application folder.

Load CSV: Loads history from the calculation_history.csv file.
